# 💸 Bill Splitter with Cheapskate Detection

## Compile & Run

```bash
g++ bill_splitter.cpp -o bill_splitter
./bill_splitter
```

## Features
- Total bill equally divide karta hai
- Har person ka paid amount track karta hai
- Cheapskate Score (0-100) deta hai
- Settlement batata hai kaun kise paisa de

## Cheapskate Score
| Score | Title |
|-------|-------|
| 0     | ✅ Gentleman/Lady |
| 1-10  | 😏 Thoda Kam Diya |
| 11-25 | 🤨 Suspicious Banda |
| 26-50 | 😤 Kanjoos |
| 51-75 | 😡 Bada Kanjoos |
